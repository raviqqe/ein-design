# v1

## Main functions

```
import Effects

main : Executors -> Effect
main executors =
  writeFile "bar.txt" (executors.execute (Effects.readFile "foo.txt"))
```

## Types

### Executors

```
type Executors =
  { execute : Effect -> EffectResult
  , rest : "readFile" -> ReadFileExecutor | "writeFile" -> WriteFile
  }
```

### Effects

```
type EffectType =
    "readFile"
  | "writeFile"
  | "getArguments"
  | "getEnvironmentVariables"
  | ...

type Effect =
    ReadFile
  | WriteFile
  | ...

type ReadFile =
  { type : "readFile"
  , filename : String
  }

type WriteFile =
  { type : "writeFile"
  , filename : String
  , content : String
  }

...
```
