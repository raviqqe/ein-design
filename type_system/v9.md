# v9

- Nominal typing
- No type aliases

## Built-in types

### Scalar types

```
Boolean
None
Number
String
```

### Function types

```
a -> b
```

### Top types

```
Any
```

## User-defined types

- Only type names are visible to other modules.
- RHS must be record or variant types.

### Record types

```
type Person = { name : String, age : Number }
```

### Union types

- No statically typed enumeration
  - Like Go

```
type Foo =
    Bar
  | Baz
  | Blah
```

## Result types

```
type Error =
  { error : None // Error indicator field
  , context : Any
  }

type Result =
    Foo
  | Error
```

### ! operator

```
foo : String -> Foo | Error
foo x =
  baz (! bar x) (! error "fail always")
```

## Option types

```
type Option =
    Foo
  | None
```

## Case expression

```
case foo of
  { name = "John" } -> e1
  { name = "Doe" } -> e2
  { name } -> e3
```

### Type switch

```
case bar of
  Foo -> e1
  Bar -> e2
  Baz -> e3
```
